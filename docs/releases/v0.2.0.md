# v0.2.0 发布说明

**发布日期**：2026-04-29

## 🎉 新功能

### 手势坐标调试面板
- **长按标题 3 秒**进入调试模式
- **4 个滑块**实时调整手势坐标：
  - 横向位置（centerXPct）
  - 上滑起点（swipeTopYPct）
  - 下滑起点（swipeBottomYPct）
  - 暂停位置（tapYPct）
- **保存/重置/测试**按钮，配置立即生效
- **无需重新编译**即可适配不同手机和 App 版本

### 长时间无命令自动停止
- 默认 **30 分钟**无命令自动停止服务
- 防止忘记关闭导致电量耗尽
- 超时时长可通过 DataStore 配置

## 🏗️ 架构改进

### DataStore 配置持久化
- 引入 Jetpack DataStore 管理所有配置
- 支持持久化：
  - 手势坐标（centerXPct, swipeTopYPct, swipeBottomYPct, tapYPct）
  - 超时时长（idleTimeoutMinutes）
  - 目标 App 列表（targetPackages）
- 响应式配置读取（Flow）

### 多 App 支持架构重构
- 创建 `SupportedApp` 数据类抽象 App 配置
- 支持为每个 App 配置独立的 `GestureConfig`
- 从硬编码 `Set<String>` 迁移到 `List<SupportedApp>`
- 为未来适配视频号/小红书打好基础

## 🧹 技术债清理

### 清理废弃资源
- 删除 4 个废弃的提示音文件（fb_*.ogg）
- 减少 APK 体积约 10KB
